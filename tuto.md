📝 Phase 1 : La préparation
Ouvrez VS Code.

Allez dans Fichier > Nouveau fichier texte.

Enregistrez-le immédiatement (Fichier > Enregistrer sous...) en lui donnant un nom terminé par .md (par exemple : article-wampum.md). C'est ce qui active la magie du logiciel.

✍️ Phase 2 : La rédaction (Le formatage)
Écrivez votre texte naturellement. Pour la mise en forme, vous avez le choix : utilisez la barre d'outils en haut de l'écran (le B pour le gras, le H pour les titres), ou tapez directement les symboles au clavier :

Titres : Ajoutez des dièses et un espace (# Titre 1, ## Titre 2).

Gras : Entourez le mot de deux étoiles (**mot**).

Italique : Entourez le mot d'une seule étoile (*mot*).

Listes : Utilisez un tiret et un espace (- point 1).

Paragraphes : Laissez toujours une ligne vide entre deux blocs de texte.

👁️ Phase 3 : La prévisualisation
Pour voir à quoi ressemble votre article :

Faites un clic droit n'importe où dans votre texte brut.

Choisissez Markdown Preview Enhanced: Open Preview to the Side.

Votre écran se coupe en deux : à gauche votre brouillon, à droite le résultat final, beau et propre.

⚙️ Phase 4 : L'exportation en HTML
Quand votre article historique est terminé et relu :

Faites un clic droit du côté de la page blanche (l'aperçu).

Cliquez sur HTML, puis sur HTML (offline).

Le logiciel génère silencieusement un nouveau fichier terminé par .html dans votre dossier.

🚀 Phase 5 : L'intégration sur votre site
Regardez dans la colonne de gauche de VS Code (là où se trouvent tous vos autres fichiers comme 01-spiritualite-nature...).

Cliquez sur le nouveau fichier .html qui vient d'être créé.

Descendez un peu pour ignorer les premières lignes de réglages techniques (les balises <head>, <style>).

Copiez tout le bloc de texte qui correspond à votre article (celui qui commence avec vos <h2> et <p>).

Collez ce code parfaitement formaté directement dans l'infrastructure de votre site web
